Bloodlink

  • Client: GameGhor
  • Time: 2025
  • Roles: web design,
    development,
    WebApp
Visit site ↗

Bloodlink is a web application designed to connect blood donors with recipients in need. The platform streamlines the process of finding compatible donors, making it easier and faster to save lives. Built with a focus on user experience, Bloodlink offers secure registration, real-time matching, and location-based search features for both donors and recipients.
